FAQ

The virtual MIDI port doesn't appear in my DAW

Make sure MSC2OSC is running and "Virtual MIDI Input" is enabled in Settings. Some DAWs require a restart to detect new MIDI destinations.

How do I configure the OSC output path?

Go to Settings > OSC Output. Use tokens like {command}, {cue}, {list} in your path template. Example: /msc/{command}/cue/{cue}

What MSC commands are supported?

MSC2OSC supports all standard MSC commands including GO, STOP, RESUME, TIMED_GO, LOAD, SET, FIRE, ALL_OFF, RESTORE, RESET, GO_OFF, and more.

Can I send OSC to multiple destinations?

Currently MSC2OSC sends to a single OSC destination. For multiple destinations, you can use an OSC routing application to redistribute the messages.
